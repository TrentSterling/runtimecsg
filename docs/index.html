Boolean Operations

Combine brushes with three CSG operations to carve, merge, and intersect geometry.

Union

A + B

Merge shapes together. Add geometry to build up your scene.

Subtract

A - B

Carve holes and cut shapes. Remove geometry from existing brushes.

Intersect

A ∩ B

Keep only the overlap. Create complex shapes from simple primitives.

Features

✦

Per-Brush CSG Engine

Chisel-style algorithm processes each brush independently. Fragments are split and evaluated against the full boolean chain for correct results.

▦

Chunk System

World-space grid chunking with dirty tracking. Only affected chunks rebuild when a brush moves or changes. Time-sliced for smooth framerates.

▶

Runtime Ready

Not editor-only. Generate and modify CSG geometry at runtime for procedural levels, destructible environments, or player-built worlds.

⌖

Editor Toolbar

Scene overlay with quick-add buttons, mode toggle (add/subtract), keyboard shortcuts, wireframe visualization, and face drag handles.

✶

Double Precision Math

All plane math uses doubles internally. No precision collapse at world-scale distances. Float conversion only at mesh output.

⬢

Brush Primitives

Box, wedge, cylinder, arch, and sphere out of the box. Plane-defined, so custom shapes are just a list of planes.

⧉

Coplanar Handling

Correct z-fighting prevention via later-brush-wins tiebreaker. No duplicate surfaces or flickering between overlapping faces.

≣

143 Unit Tests

Comprehensive test coverage for all boolean operations, edge cases, polygon splitting, categorization, and chain evaluation.

⌂

UPM Package

Install via Unity Package Manager from git URL. Clean separation of runtime and editor assemblies. Unity 2021.3+.

How It Works

Chisel-style per-brush CSG in four steps.

Split

Each brush's face polygons are split against the planes of all overlapping brushes, creating convex fragments.

Categorize

Each fragment is classified as inside, outside, or coplanar relative to every other brush using centroid sampling.

Evaluate

The boolean chain (union, subtract, intersect) is evaluated for both sides of each fragment to determine if it's on a visible boundary.

Build

Visible fragments are collected, clipped to chunk bounds, and built into optimized meshes with colliders.
