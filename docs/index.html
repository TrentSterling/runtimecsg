RuntimeCSG

Non-destructive CSG brush modeling for Unity. Boolean operations in real-time, in-editor and at runtime.

GitHub Install

Boolean Operations

Union, subtract, and intersect brushes. Chisel-style per-brush engine evaluates each brush independently.

Brush Primitives

Box, wedge, cylinder, arch, and sphere primitives ready to go. Combine them to build anything.

Runtime Ready

Not just an editor tool. Generate and modify CSG geometry at runtime for procedural levels and player-built worlds.

Chunk System

Grid-based spatial chunking with dirty tracking. Only affected chunks rebuild when a brush changes.

Editor Tools

Scene overlay toolbar, wireframe visualization, and face handles for direct brush manipulation.

Double Precision

Plane math uses doubles internally. No precision collapse at world-scale distances.

Install via Package Manager

https://github.com/TrentSterling/runtimecsg.git
